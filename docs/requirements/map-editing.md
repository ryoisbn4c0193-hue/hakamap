# 地図編集機能要件

## 背景

- 墓地全体の図面を、利用者が所有する画像またはPDFから取り込めるようにする。
- 背景取り込みの詳細は`docs/design/map-background-import.md`に従う。
- 背景画像を移動、拡大、縮小、回転、および差し替えできるようにする。
- X方向とY方向を独立して拡大または縮小できるようにする。
- 背景画像は任意角度で回転できるようにする。
- 背景調整時にガイド線を表示できるようにする。
- 背景調整時のスナップをONまたはOFFにできるようにする。

## 墓所

- 墓所は一点を示すピンではなく、長方形として地図上に配置する。
- MVPでは墓所の形状を長方形だけとし、多角形は将来拡張とする。
- 墓所の位置、大きさ、および回転角度を地図座標として保存する。
- 墓所の初期回転角度は0度とする。
- MVPでは墓所を回転するUIを提供しない。
- 背景画像を移動または再調整しても、墓所の保存座標を自動的に変更しない。
- 墓所ラベルは、管理番号、墓所名、管理番号と墓所名の両方、または非表示から選択できるようにする。
- 地図上で墓所を削除するときは確認画面を表示し、確認後に即時削除する。
- 墓所の単体作成、一括作成、および指定範囲を埋める連続生成を行えるようにする。
- 墓所を移動、コピー、削除、およびサイズ変更できるようにする。
- 墓所のサイズは、数値入力とドラッグ操作の両方で変更できるようにする。
- 墓所を複数選択し、一括移動、一括コピー、および一括削除できるようにする。
- 墓所同士の重なりは禁止し、保存時に検証エラーを表示する。
- 墓所同士の辺または角が接する配置は許可する。
- 墓所をコピーするときは管理番号を引き継がず、コピー後の墓所を情報未完成状態とする。
- 複数の墓所へ管理番号を一括採番する場合は、左から右、次に上から下の順で採番する。
- 墓所ごとに、複数行の自由入力が可能な備考欄を設ける。
- 備考は保存、バックアップ、インポート、およびエクスポートの対象とする。

## エリア

- MVPではエリアを地図上の長方形の区画として描画し、多角形は将来拡張とする。
- エリアを作成、移動、サイズ変更、および名称変更できるようにする。
- 地図上のエリアは、プロジェクトごとのエリアマスターと関連付ける。
- 異なるエリアの長方形同士が重なる配置は禁止し、保存時に検証エラーを表示する。
- エリアの長方形は作成後も移動およびサイズ変更できるようにする。
- 形状変更によって関連する墓所がエリア範囲外になった場合は、利用者へ警告する。
- 墓所のエリアは位置から自動判定し、利用者による手動上書きを設けない。
- どのエリアにも属さない墓所は未割当として警告する。
- 範囲外であることを理由に保存を禁止せず、利用者は警告を確認したうえで登録できる。
- エリアごとにプリセットから色を選択できるようにする。
- エリアごとに異なる境界線と塗りつぶしを表示できるようにする。
- エリア名を地図上に表示できるようにする。
- エリアごとに表示と非表示を切り替えられるようにする。

## 地図要素

- 初期状態では、墓所、通路、入口、および建物（本堂）を配置できるようにする。
- 初期状態で使用する形状は次のとおりとする。
  - 墓所: 長方形
  - 通路: 折れ線に幅を持たせた形状
  - 入口: 地図上の一点を示すアイコン
  - 建物（本堂）: 長方形または多角形
- 地図要素の種類はプロジェクトごとのマスターとして管理する。
- 利用者が地図要素の種類を追加できるようにする。
- 種類を追加するときは、長方形、多角形、線、またはアイコンから形状を選択できるようにする。
- 種類の名称変更、並べ替え、利用停止、および再有効化に対応する。
- 種類の名称変更は、その種類を参照するすべての地図要素の表示へ自動的に反映する。

## スナップ

- スナップ機能をONまたはOFFに切り替えられるようにする。
- スナップ対象は、墓所の辺と角、およびエリアの辺と角だけとする。
- グリッドへのスナップはMVPに含めない。
- スナップ時はガイド線を表示する。
